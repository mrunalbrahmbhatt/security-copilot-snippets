# Change Log

## [1.0.0] - 2026-04-03

### Added
- YAML snippets for Security Copilot plugin manifests (API, KQL, GPT, LogicApp, Agent)
- OpenAPI 3.0 specification snippets for Security Copilot API plugins
- Pre-built KQL query templates (risky sign-ins, impossible travel, device details, geo sign-ins)
- Command palette commands for inserting all template types
- Quick-pick template menu via "Security Copilot: Show All Templates"
- Right-click context menu in YAML files
- Support for all authentication types (AAD, ApiKey, OAuth, Basic, ServiceHttp)
- Interactive agent manifest template
- Pre-built patterns for IP reputation, email via Logic App
